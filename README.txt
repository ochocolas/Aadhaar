#Proyecto de Privado

Sistema para registro de documentos de identificación basado en requerimientos en la India.

Consiste en un app desktop para registro en lugares remoto que sincronizan a un sitio principal web. 
Incluye generación de nombres aleatorios para tener algo de data para probarlo.

Requiere Devexpress ( no recuerdo la versión pero ahí debe de estar en el código, lo siento)
